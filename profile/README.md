## 繁體中文

**企業資訊委外與電腦維護的專家**

### 關於我們

群兆資訊有限公司（Megapower Asia LLC）自 2008 年成立於台北，是一家服務導向的資訊團隊，專注於**企業資訊委外與電腦維護**。深耕台灣中小企業資訊服務超過 18 年，累計服務 300+ 家企業客戶，產業涵蓋製造、貿易、醫療、教育、設計、新創等多元領域。

群兆的服務哲學是 **ClientCentric IT — 以客戶業務需求為核心的 IT 服務**：先理解您的工作流程與營運目標，再規劃涵蓋電腦維護、資訊委外、雲端應用、資安防護與備份保全的整體資訊策略，讓技術投資始終對齊業務目標。我們把資訊環境維護好，讓企業專注在核心業務。

服務對象為 1–200 人規模的台灣中小企業，特別擅長 15–100 人規模企業的整體資訊委外服務。

### 服務領域

| 服務 | 內容 |
|------|------|
| **資訊委外與電腦維護** | 長期擔任企業的資訊合作夥伴，承接日常電腦維護、設備汰換、帳號與權限管理及使用者支援，並提供虛擬資訊長（vCIO，virtual CIO）顧問與軟硬體採購服務。 |
| **企業基礎架構與雲端** | 提供本地端基礎設施即服務（IaaS，Infrastructure as a Service）與雲端訂閱方案，建置伺服器、儲存與網路設備，規劃 Microsoft 365 雲端遷移、企業郵件代管與網域名稱（DNS）代管。 |
| **異地備份與資料保全** | 依企業風險等級規劃本地備份與異地備援，落實 3-2-1 備份原則，以國際級電信機房為備援據點，並定期執行還原測試。 |
| **企業資訊安全** | 整合防毒、次世代安全營運中心（SOC，Security Operations Center）監控、資安健檢、弱點掃描與滲透測試、社交工程演練、資安顧問與事件應變，依 NIST 網路安全框架（NIST CSF）規劃防護層級。 |
| **數位轉型與企業顧問** | 協助導入遠端辦公、雲端協作與內部流程數位化，並以企業顧問與陪跑教練的形式，分階段陪同企業推進改善。 |

### 服務承諾

- **ClientCentric IT**：以客戶業務需求為起點規劃資訊策略，讓技術佈署始終對齊業務目標。
